  • 什么是“本息保障”计划？

    保障投资人在天天聚财投资理财的本金和利息收益安全

    天天聚财推出“本息保障计划”，保障投资人在天天聚财投资理财的本金和利息收益安全。
    "本息保障计划”：是指在天天聚财投资的投资者其投资本金及利息都受到保护，不会因为借款人逾期、不还款等违约行为而受到损失。

  • 学会五招，本息有保障 !

    出现坏账，偿付投资人剩余本金及利息；
    出现逾期，担保履约还未完成时，投资人不会承担任何损失；
    天天聚财将会在逾期当天使用“风险备用金”向理财人垫付此笔借款的逾期本息 。
    天天聚财自动通知托管平台将回款转入投资人的帐号
    如借款期限届满之日借款者不能偿还投资者本金及利息，担保机构将按照本合同约定向投资者偿付借款的本金和利息。
    担保机构提供100%本息连带责任担保，完全保障投资人本金及理财利息 ;
    所有天天聚财投资人的投资本金都将自动获得本息保障，无需特别加入或任何费用 。
    风险备用金账户，担保公司全额本息担保！

    天天聚财的优势在于可以提供丰富的投资项目，让投资人看到真实、透明的借款信息，进行自由选择以组成最合您心意的专有投资组合。即便在有本金保障的情况下，一旦借款人不能按时偿还本金，触发本金保障垫付前的资金回收延迟会降低您的整体资金利用效率，影响您的加权收益率。因此，为保证更有效地利用资金，我们建议您积极自主地判断项目风险并挑选投资项目。

  • 天天聚财—银行级别的投资保障

    保障投资人在天天聚财投资理财的本金和利息收益安全

    风险备用金账户

    天天聚财设立“风险备用金账户”，为“本息保障计划”提供资金保障。“风险备用金账户”资金来源：借款人融资金额中收取0.5%-1%作为风险备用金；担保公司担保费中收取30%作为风险备用金；借款人逾期违约金计入风险备用金账户。

    担保公司全额本息担保

    天天聚财平台任何一笔借款都获得担保公司全额本息担保。当借款人出现逾期、违约时由担保公司垫付全部本金和利息。担保公司全额本息担保是保障投资人利益和平台安全的核心举措。

    反担保措施

    任何借款人都会向担保公司提供反担保方案，包括但不限于不动产抵押（例如：房产抵押、厂房抵押、土地抵押）、机动车抵质押、动产抵押质押、设备机械质押、权利质押等各种反担保措施。

    成熟可信的收益模式

    天天聚财联合担保公司做个人和中小微企业中短期借款业务，操作方式成熟，经营丰富。天天聚财收取借款人融资服务费用，不承担借款人违约风险，保证平台独立安全运营。担保公司推荐标的，提供担保服务，承担违约风险。

  • 天天聚财平台提供居间撮合服务的合法性

    根据《合同法》第23章关于“居间合同”的规定，特别是第424条规定的“居间合同是居间人向委托人报告订立合同的机会或者提供订立合同的媒介服务，委托人支付报酬的合同”，天天聚财平台作为合法设立的中介服务机构，为民间借贷提供撮合，使借贷双方形成借贷关系并收取相关报酬的居间服务有着明确的法律基础。

  • 理财人及借款人双方民间借贷关系的合法性

    根据《合同法》第196条规定“借款合同是借款人向贷款人借款，到期返还借款并支付利息的合同”，《合同法》允许自然人等普通民事主体之间发生借贷关系，并允许出借方到期可以收回本金和符合法律规定的利息。

  • 理财人通过天天聚财平台获得的出借理财收益的合法性

    根据最高人民法院《关于人民法院审理借贷案件的若干意见》第6条：“民间借贷的利率可以适当高于银行的利率，各地人民法院可以根据本地区的实际情况具体掌握，但最高不得超过银行同类贷款利率的四倍（包含利率本数）。超出此限度的，超出部分的利息不予保护。”天天聚财的出借理财收益未超过银行同类贷款利率的四倍，为合法利息收益，受到法律保护。

  • 理财人及借款人双方民间借贷关系的合法性

    根据《合同法》第196条规定“借款合同是借款人向贷款人借款，到期返还借款并支付利息的合同”，《合同法》允许自然人等普通民事主体之间发生借贷关系，并允许出借方到期可以收回本金和符合法律规定的利息。

  • 保障一：融资人信息公开透明

    所有尽职调查信息都会被公开，投资人在投资前就能对项目有100%的掌控。

    保障二：天天聚财不经手投资人资金

    我们不集资金池，资金的流转不会经过平台，平台也不会有任何权限去支配资金。资金的每一次流动都会被投资者清晰的监控。天天聚财与国内知名第三方支付公司宝付和汇潮支付合作 ，保证您在资金运作时的交易安全和信息安全

  • 保障三：资金流转信息透明

    投资后，资金的所有流转使用信息都能可查，项目将拥有上市公司般的信息透明度，投资人也将拥有最完整的知情权。

    保障四：平台透明

    以中小微企业和个人提供综合性的金融服务为核心业务。以信誉为基本的互联网网贷平台，平台数据信息皆为真实可靠。

  • 第一道审核：

    贷款机构核实包括获取、鉴别、审核借款客户的个人信息，征信报告，联系人信息，工作信息，银行流水，实地验证，财产信息等信用信息。然后由独立的风控团队独立审核，项目才可以通过第一道审核。

    第二道审核:

    天天聚财将所推荐的借款客户的所有资料发至合作担保公司进行终审。合作担保公司经验丰富的审核团队会分析该借款人的个人信息、征信信息、财产信息、工作/公司信息等对还款能力及还款意愿的判断最为显著的变量，决定是否对该项目提供担保。

  • 借款项目审核流程明细

    实地考察：担保公司实地考察线下借款人

    资料审核：审核包括银行流水、资产情况等十余项必备材料

    背景调查：对借款人背景、联系人信息详尽调查

    还款能力：结合资料审查和数据分析综合评估其还款能力

    资产评估：严格评估抵押/质押的资产状况

    抵押办理：资产通过估值后办理相关抵押手续

    会议决定：我们会根据融资人的具体情况制定出一整套安全可行的融资方案。只有收益和风险最佳的项目共放在平台上供投资人选择。

  • 隐私安全

    我们郑重承诺并尊重、保护用户的个人隐私，未经用户授权我们不会公布与用户个人有关的资料，除非有法律程序要求。我们将定期检查隐私保护措施，跟踪安全技术的发展趋势，更新技术和管理措施。

    只有相关工作人员才可以接触到用户的信息。

    资金安全

    天天聚财与国内知名第三方支付公司宝付以及汇潮支付合作，保证您在资金运作时的交易安全和信息安全。

  • 杜绝线下交易

    天天聚财真诚的建议您避免线下交易，因为民间借贷所涉及的借款合同、信用审核等程序都存在众多专业的法律和技术环节，如果您并非专业人士，很容易在其中某一环节疏忽或收到虚假信息，从而形成非常高的逾期风险，甚至遭遇诈骗或其他犯罪侵害

    系统安全

    为了保障您在天天聚财平台的正常运作，避免账户遭他人恶意盗用造成的个人信息外泄或资金损失，除了平台本身提供的安全防护措施以外，我们建议您提升个人电脑的安全级别以及自身的安全防范意识。
